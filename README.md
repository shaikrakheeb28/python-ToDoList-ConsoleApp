# To-Do List App (Console)

This is a simple console-based To-Do List application written in Python.

## Features

- Add tasks
- View all tasks with status (Pending/Done)
- Mark tasks as done
- Delete tasks

## Steps to Run in PyCharm IDE

1. Open PyCharm IDE.
2. Click on `File` > `Open` and select the `ToDoListApp` directory.
3. Run `to_do_list.py` to start the application.
4. To run test cases, right-click on `test_to_do_list.py` and choose `Run`.

## Testing

- Uses `unittest` to test core functionalities:
  - Adding a task
  - Marking a task as done
  - Deleting a task
